# HACKATHON.md

## Your Mission
Build a web application! Start from the Next.js template and make it yours.

## Getting Started
```bash
npm install
npm run dev      # runs on port 3000
npm run build    # production build
```

## Ideas
- Dashboard for a public API
- Markdown editor with live preview
- Link shortener with analytics
- AI prompt gallery

## Judging Criteria
- **Functionality** (30%)
- **Design** (30%)
- **Code Quality** (20%)
- **Creativity** (20%)
